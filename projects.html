Gridiron Guru

Create statistical models capable of predicting fantasy football performance. Utilize these models in an application that will recommend drafting and weekly lineup strategies for users.

Models were created using Jupyter python notebooks. The GUI for the application was designed using Vue.js. The application runs using electron. All player statistical data and user fantasy team info is stored on a NoSQL database hosted on firebase.

  • Languages: Javascript (Vue.js), Python
  • Date: January 2017 - June 2017

Personal Affiliation Extraction

Extract the most closely assorted affiliation for an individual from an utterance. Utterances are provided by Digital Democracy.

Potential affiliations are stored in an inverted index. A feature vector is extracted for each affiliation found in every utterance. A score is calculated based on the feature vector. The utterance is assigned the affiliation with the highest score

  • Languages: Python
  • Date: May 2017 - June 2017

Stanchef's Kitchen

Website built to handle online ordering for a fake restaurant.

Uses a PostgreSQL database to hold account, menu, and order information. Database hosted on a heroku server. Frontend is designed using JavaServerFaces (xhtml). Java controllers handle input from users to make requests and update the database.

  • Languages: Java, PostgreSQL
  • Date: May 2017 - June 2017
